# Tests

## Backend (Japa)

Script: `pnpm test` (alias `node ace test`).
Répertoire: `tests/` (unit/functional, selon la suite).

## Frontend Inertia (Vitest)

Script: `pnpm test:inertia` (alias `vitest run`).
Répertoire: `tests/inertia` (selon la structure du repo).

## Typecheck / lint

- `pnpm typecheck`
- `pnpm lint`
